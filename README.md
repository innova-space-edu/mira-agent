# MIRA Agent (Web)

Base web para construir **MIRA**, un agente de IA conversacional con:
- Chat moderno
- Ventana de tareas (modo agente)
- Estados del agente (IDLE/PLANNING/EXECUTING/etc.)
- Listo para conectar un cerebro IA cloud en el Paso 2

## Publicación en GitHub Pages
1. Settings → Pages
2. Source: Deploy from a branch
3. Branch: `main`
4. Folder: `/ (root)`
5. Guardar

Luego abre la URL:
`https://TU_USUARIO.github.io/mira-agent/`

## Paso 2
Conectar cerebro IA cloud (backend seguro) para:
- Conversación real
- Tool calling
- Ejecución de tareas web
